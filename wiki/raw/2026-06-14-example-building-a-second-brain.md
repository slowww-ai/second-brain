# Building a Second Brain — notes

Tiago Forte's premise: our minds are for *having* ideas, not *holding* them. A
"second brain" is an external, trusted system where you offload what you learn so
you can think with it later instead of trying to remember it.

## The CODE method

1. **Capture** — keep what resonates. Don't try to save everything; save what
   sparks something.
2. **Organize** — file for action, not for storage. Ask "which project does this
   serve?" rather than "which category does this belong to?"
3. **Distill** — find the essence. Highlight the highlights so a future,
   busier you can grasp a note in seconds.
4. **Express** — turn notes into something: a brief, a post, a decision. Output
   is the point; a vault you never use is just hoarding.

## Intermediate packets

Work is made of reusable "intermediate packets" — a chart, a paragraph, a
checklist. Capture and refine these and future projects assemble faster because
you're recombining, not starting from a blank page.

## Why it works

Capturing reduces the anxiety of forgetting. Distilling turns raw material into
something usable. Expressing closes the loop — knowledge that produces output is
the only knowledge that compounds.
